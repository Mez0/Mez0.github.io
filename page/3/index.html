在 OS X 上使用 aria2 下载百度云盘上的资源

Posted on 2015-08-17   |   visitors

在国内网络环境下，基本逃不开要在百度云盘这个傻逼平台上下点东西。而由于目前为止 OS X 下还没有类似 Windows 平台下的云管家客户端，再加上 Chrome 自带下载百度云时速度十分无力，最操蛋的是貌似最近「直接将下载地址拷贝到迅雷」的方法也不好用了，因此整个下载速度就呵呵了。所以在网上找到了这样一个跨平台的命令行下载工具 aria2，配合一个 Chrome 插件，算是让下载云盘不再那么蛋疼了。

  • 在 Terminal 中输入brew install aria2来安装 aria2
  • 安装 Chrome 插件
  • 下载时，将插件生成的指令复制到 Terminal
  • 文件会下载到当前工作目录
  • Done
